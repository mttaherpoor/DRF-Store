# 🏪 DRF Store

A Django REST Framework API for managing products, users, and orders.  
Includes authentication, CRUD endpoints, and Docker setup for easy deployment.

## 🚀 Features
- Product & category management  
- JWT authentication  
- PostgreSQL support
